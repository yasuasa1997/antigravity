# 📈 Stock Price Dashboard

リアルタイムで株価を自動収集し、ブラウザ上でインタラクティブに可視化するダッシュボードアプリケーションです。

## 🎯 特徴

- **自動データ収集** — Yahoo Finance API から 15秒間隔で株価を自動取得
- **インタラクティブなチャート** — Plotly によるズーム・ホバー対応の株価推移グラフ
- **銘柄の動的管理** — ダッシュボードのUIから銘柄の追加・削除が即座に反映
- **企業名の自動取得** — ティッカーコードに加えて企業名を自動表示
- **日本株・米国株対応** — `7203.T`（トヨタ）や `AAPL`（Apple）など幅広い銘柄に対応

## 🛠 技術スタック

| 項目 | 技術 |
|---|---|
| 言語 | Python |
| UI | [Streamlit](https://streamlit.io/) |
| データ取得 | Yahoo Finance API（`requests` 経由） |
| データ保存 | SQLite |
| グラフ描画 | [Plotly](https://plotly.com/) |

## 📁 ファイル構成

```
stock_dashboard/
├── app.py              # Streamlit ダッシュボード（メインUI）
├── collector.py        # 株価データ自動収集スクリプト
├── database.py         # SQLite データベース操作モジュール
├── stock_api.py        # Yahoo Finance API 連携モジュール
├── config.py           # 設定ファイル（銘柄リスト・取得間隔・DB名）
├── requirements.txt    # 依存パッケージ一覧
├── start_dashboard.bat # ワンクリック起動用バッチファイル
└── stocks.db           # SQLite データベース（自動生成）
```

## 🚀 セットアップ

### 前提条件

- Python 3.8 以上
- pip（Python パッケージマネージャー）
- インターネット接続

### インストール

```bash
cd stock_dashboard
pip install -r requirements.txt
pip install streamlit
```

## 📖 使い方

### 方法1: バッチファイルで一括起動（推奨）

`start_dashboard.bat` をダブルクリックするだけで、依存パッケージのインストール・データ収集・ダッシュボードがすべて起動します。

### 方法2: 手動で起動

**ステップ1: データ収集を開始**

```bash
python collector.py
```

別のターミナルを開いてください。

**ステップ2: ダッシュボードを起動**

```bash
streamlit run app.py
```

ブラウザが自動的に開き、ダッシュボードが表示されます（通常 http://localhost:8501）。

## 🖥 ダッシュボードの操作

### 銘柄の追加

1. サイドバーの **「Add Ticker」** 欄にティッカーコードを入力（例: `7203.T`）
2. **「Add」** ボタンをクリック
3. 企業名が自動取得され、ダッシュボードに即時反映

### 銘柄の削除

1. サイドバーの **「Remove Ticker」** ドロップダウンから銘柄を選択
2. **「Remove」** ボタンをクリック

### 表示銘柄の切り替え

- サイドバーの **「Select Tickers to View」** で表示したい銘柄を選択/解除

### 自動更新

- サイドバーの **「Auto-refresh (15s)」** チェックボックスをONにすると、15秒ごとにダッシュボードが自動更新されます

## ⚙️ 設定

`config.py` で以下の設定を変更できます:

| 設定項目 | デフォルト値 | 説明 |
|---|---|---|
| `TICKERS` | `["AAPL", "MSFT", "7203.T", "9997.T"]` | 初期監視対象銘柄（UIからも追加/削除可能） |
| `INTERVAL_SECONDS` | `15` | データ取得間隔（秒） |
| `DB_NAME` | `"stocks.db"` | SQLite データベースファイル名 |

## 📌 ティッカーコードの例

| コード | 企業名 | 市場 |
|---|---|---|
| `AAPL` | アップル | 米国 |
| `MSFT` | マイクロソフト | 米国 |
| `GOOGL` | アルファベット（グーグル） | 米国 |
| `7203.T` | トヨタ自動車 | 日本 |
| `9984.T` | ソフトバンクグループ | 日本 |
| `6758.T` | ソニーグループ | 日本 |
| `9997.T` | ベルーナ | 日本 |

> **💡 日本株のティッカー**: 証券コード + `.T`（例: トヨタ → `7203.T`）

## ⚠️ 注意事項

- プロキシ環境下では SSL 証明書の検証を無効化（`verify=False`）しています。本番環境では適切な証明書設定をご検討ください。
- Yahoo Finance API の利用規約に従ってご使用ください。
- データ取得頻度が高すぎると、APIからレート制限を受ける可能性があります。
